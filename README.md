## Building the Engine (Linux)

### Prerequisites
- CMake >= 3.10
- SDL2, SDL2_image, SDL2_ttf, SDL2_mixer
- OpenGL
- Lua 5.4
- g++ (C++17)

### Build Steps
```bash
# Clone the repository
git clone git@github.com:basketoengine/Basketo.git
cd Basketo
mkdir build && cd build
cmake ..
make -j$(nproc)
```

### Running
```bash
./BasketoGameEngine
```

If you want to run the physics test:
```bash
./PhysicsTest
```
